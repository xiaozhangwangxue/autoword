# Contributing

Thanks for helping improve AutoWord.

1. Fork the repository and create a focused branch from `main`.
2. Do not commit virtual environments, logs, real documents, or any sensitive content.
3. Run `python -m unittest discover -s tests -v` before opening a pull request.
4. Explain the change, motivation, and validation in the pull request.

## Reporting issues

Include reproduction steps, expected and actual results, Python version, and operating system. Never attach documents containing sensitive information to a public issue.
