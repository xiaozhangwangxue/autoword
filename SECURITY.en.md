# Security policy

## Supported versions

Only the latest version on the default `main` branch is maintained.

## Reporting a vulnerability

Do not report security vulnerabilities in a public issue. Contact the repository maintainer with the impact, reproduction steps, and a possible fix; the maintainer will coordinate remediation and disclosure.

## Deployment reminder

This application processes uploaded Word documents and has no built-in authentication. Run it only on a trusted network, or configure HTTPS, authentication, rate limits, and request-size limits at a reverse proxy or gateway.
